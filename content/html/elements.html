Grundstruktur

Element Beschreibung
html Umschließt das komplette HTML-Dokument
head Kopfbereich mit Metainformation
body Hauptinhalt der Seite

Metainformationen

Element Beschreibung
base Basispfad für Links
link Verlinkt externe Ressourcen
meta Metadaten
noscript Falls Scripting nicht unterstützt wird
script Scripting
style Stylesheet
title Titel des Dokuments (im Browser-Tab)
